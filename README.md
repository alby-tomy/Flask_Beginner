# Todo App

## Installation

1. **Navigate to the Project Directory:**
   ```bash
   cd todo_app
2. **Activate the venv:**
   ```bash
   source bin/activate
3. **Run the Flask Application server.py**
   ```bash
   cd ..
   python3 server.py
### The application will be accessible at http://127.0.0.1:5000/ in your web browser.
# Basic CRUD Operations
The Flask Todo App supports basic CRUD (Create, Read, Update, Delete) operations. Here's how you can perform these operations:

## Create (Add a Todo)
- Navigate to the "Add Todo" page by clicking on the corresponding link.
- Fill in the details for the new todo and click the "Add" button.

## Read (View Todos)
- The home page displays a list of all existing todos.
- Click on a todo to view its details.

## Update (Edit a Todo)
- Navigate to the "Edit Todo" page by clicking on the edit icon next to a todo.
- Modify the todo details as needed and click the "Update" button.

## Delete (Remove a Todo)
- On the home page, each todo entry has a "Delete" button.
- Click the "Delete" button next to the todo you want to remove.
 
#### These basic CRUD operations allow you to manage your todos effectively. Customize and expand the application based on your specific requirements.
